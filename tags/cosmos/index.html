Restake support

Hi stakers!, we’ve enabled restaking support for 3ventures validators in several networks: Restake (you can click on the icons for each network to select which one you want to use), is a service that performs several tasks and provides some benefits for you: Your wallet (Keplr for example) provides an authorization for the restaking operation, that is: just claim rewards on the validator and compound the rewards into staking, nothing else.

IBC Transfers from the CLI

When you’re interacting via the CLI client with different networks, you can use it for also sending tokens, just instead of using ${CHAINCOMMAND} tx bank send you should use ${CHAINCOMMAND} tx ibc-transfer transfer transfer channel-${CHANNEL}} ${TARGET} ${AMOUNT}${TOKEN} --from ${FROM}. Remember that you can check the appropriate channel number by visiting the chain explorer and checking the list of relaying channels.
